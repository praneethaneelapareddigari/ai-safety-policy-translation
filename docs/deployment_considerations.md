## Intended structure

has property X (e.g. serves non-English users, exposes tool-calling to
untrusted input), the benchmarks suggest evaluating for Y before
shipping." Each recommendation will cite the specific evidence-synthesis
row it's grounded in — no recommendation without a citable measurement.
